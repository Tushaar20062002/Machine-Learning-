# 🚗 Car Price Prediction using Machine Learning

A Machine Learning project that predicts the selling price of used cars using regression algorithms. This project focuses on data preprocessing, feature encoding techniques, model comparison, and performance evaluation.

---

## 📌 Project Overview

The objective of this project is to build a regression model capable of predicting the selling price of a used car based on various vehicle attributes.

The notebook follows a complete machine learning workflow from data exploration to model evaluation.

---

## 📊 Dataset

The dataset contains information about used cars, including:

- Model
- Year
- Transmission
- Fuel Type
- Mileage
- Engine Size
- MPG
- Tax
- Selling Price

---

## ⚙️ Data Preprocessing

The dataset was prepared using:

- Handling missing values
- Duplicate removal
- Feature encoding
- Train-Test Split

---

## 🏷️ Encoding Techniques

Two different categorical encoding techniques were applied and compared.

### Label Encoding

Categorical features were converted into numerical values using Label Encoding.

### One-Hot Encoding

Categorical features were transformed into binary vectors using One-Hot Encoding.

The performance of both approaches was compared using multiple regression models.

---

## 🤖 Machine Learning Models

Multiple regression algorithms were trained and evaluated.

The models were compared based on their prediction performance to determine which encoding technique produced better results.

---

## 📈 Model Evaluation

Model performance was evaluated using:

- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

The notebook compares model scores for both Label Encoding and One-Hot Encoding.
